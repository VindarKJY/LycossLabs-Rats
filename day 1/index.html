Atribut Global

Atribut global dapat digunakan pada hampir semua elemen HTML:

accesskey autocapitalize class contenteditable data-* dir draggable enterkeyhint hidden id inputmode is itemid itemprop itemref itemscope itemtype lang nonce slot spellcheck style tabindex title translate

Daftar ini mencakup elemen HTML5 yang umum digunakan. Beberapa elemen mungkin memiliki atribut tambahan yang spesifik untuk penggunaan tertentu.
