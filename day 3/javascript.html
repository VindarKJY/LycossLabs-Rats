üåê Objek Global

Object
Objek dasar dalam JavaScript, semua objek lain diturunkan dari Object
Objek
const obj = { name: "John", age: 30 };
Array
Objek untuk menyimpan kumpulan nilai dalam satu variabel
Objek
const arr = [1, 2, 3, 4, 5];
Function
Objek yang merepresentasikan fungsi dalam JavaScript
Objek
function myFunc() { return "Hello"; }
String
Objek untuk merepresentasikan dan memanipulasi urutan karakter
Objek
const str = new String("Hello World");
Number
Objek pembungkus untuk nilai numerik
Objek
const num = new Number(42);
Boolean
Objek pembungkus untuk nilai true atau false
Objek
const bool = new Boolean(true);
Date
Objek untuk bekerja dengan tanggal dan waktu
Objek
const today = new Date();
Math
Objek built-in dengan properti dan metode untuk konstanta dan fungsi matematika
Objek
Math.PI; // 3.141592653589793
RegExp
Objek untuk bekerja dengan ekspresi reguler
Objek
const regex = /ab+c/;
Error
Objek untuk merepresentasikan kesalahan runtime
Objek
throw new Error("Something went wrong");
Promise
Objek yang merepresentasikan penyelesaian (atau kegagalan) operasi asinkron
Objek
new Promise((resolve, reject) => { ... });
Map
Koleksi pasangan kunci-nilai yang dapat diiterasi
Objek
const map = new Map();
Set
Koleksi nilai unik yang dapat diiterasi
Objek
const set = new Set([1, 2, 3]);

üìä Metode Array

push()
Menambahkan elemen ke akhir array dan mengembalikan panjang array baru
Metode
arr.push(6); // [1,2,3,4,5,6]
pop()
Menghapus elemen terakhir dari array dan mengembalikannya
Metode
arr.pop(); // 5, array menjadi [1,2,3,4]
shift()
Menghapus elemen pertama dari array dan mengembalikannya
Metode
arr.shift(); // 1, array menjadi [2,3,4,5]
unshift()
Menambahkan elemen ke awal array dan mengembalikan panjang array baru
Metode
arr.unshift(0); // [0,1,2,3,4,5]
concat()
Menggabungkan dua atau lebih array dan mengembalikan array baru
Metode
arr1.concat(arr2);
slice()
Mengembalikan salinan dangkal dari bagian array
Metode
arr.slice(1, 3); // [2,3]
splice()
Mengubah konten array dengan menghapus atau mengganti elemen
Metode
arr.splice(2, 1, 'a'); // [1,2,'a',4,5]
forEach()
Menjalankan fungsi sekali untuk setiap elemen array
Metode
arr.forEach(item => console.log(item));
map()
Membuat array baru dengan hasil memanggil fungsi pada setiap elemen
Metode
arr.map(x => x * 2); // [2,4,6,8,10]
filter()
Membuat array baru dengan semua elemen yang lulus tes
Metode
arr.filter(x => x > 2); // [3,4,5]
reduce()
Menerapkan fungsi terhadap akumulator dan setiap elemen
Metode
arr.reduce((acc, val) => acc + val, 0); // 15
find()
Mengembalikan nilai elemen pertama yang memenuhi kondisi
Metode
arr.find(x => x > 3); // 4
some()
Menguji apakah setidaknya satu elemen lulus tes
Metode
arr.some(x => x > 4); // true
every()
Menguji apakah semua elemen lulus tes
Metode
arr.every(x => x > 0); // true
sort()
Mengurutkan elemen array dan mengembalikan array
Metode
arr.sort((a, b) => a - b);
reverse()
Membalikkan urutan elemen dalam array
Metode
arr.reverse(); // [5,4,3,2,1]

üî§ Metode String

charAt()
Mengembalikan karakter pada indeks tertentu
Metode
"Hello".charAt(1); // "e"
concat()
Menggabungkan dua atau lebih string
Metode
"Hello".concat(" World"); // "Hello World"
includes()
Memeriksa apakah string berisi substring tertentu
Metode
"Hello".includes("ell"); // true
indexOf()
Mengembalikan indeks kemunculan pertama substring
Metode
"Hello".indexOf("l"); // 2
replace()
Mengganti kemunculan substring dengan string baru
Metode
"Hello".replace("l", "x"); // "Hexlo"
slice()
Mengekstrak bagian string dan mengembalikannya sebagai string baru
Metode
"Hello".slice(1, 3); // "el"
split()
Membagi string menjadi array substring
Metode
"Hello World".split(" "); // ["Hello", "World"]
substring()
Mengembalikan bagian string antara indeks awal dan akhir
Metode
"Hello".substring(1, 3); // "el"
toLowerCase()
Mengubah string menjadi huruf kecil
Metode
"HELLO".toLowerCase(); // "hello"
toUpperCase()
Mengubah string menjadi huruf besar
Metode
"hello".toUpperCase(); // "HELLO"
trim()
Menghapus spasi dari kedua ujung string
Metode
" Hello ".trim(); // "Hello"
startsWith()
Memeriksa apakah string dimulai dengan substring tertentu
Metode
"Hello".startsWith("He"); // true
endsWith()
Memeriksa apakah string diakhiri dengan substring tertentu
Metode
"Hello".endsWith("lo"); // true

üì¶ Metode Object

Object.keys()
Mengembalikan array dari properti enumerable milik objek
Metode
Object.keys({a:1, b:2}); // ["a", "b"]
Object.values()
Mengembalikan array dari nilai properti enumerable milik objek
Metode
Object.values({a:1, b:2}); // [1, 2]
Object.entries()
Mengembalikan array dari pasangan [key, value] milik objek
Metode
Object.entries({a:1, b:2}); // [["a",1], ["b",2]]
Object.assign()
Menyalin nilai dari satu atau lebih objek sumber ke objek target
Metode
Object.assign({}, {a:1}, {b:2}); // {a:1, b:2}
Object.freeze()
Membekukan objek, mencegah modifikasi properti baru
Metode
Object.freeze(obj);
Object.seal()
Menyegel objek, mencegah penambahan/hapus properti
Metode
Object.seal(obj);
Object.create()
Membuat objek baru dengan prototipe yang ditentukan
Metode
Object.create(proto);

Fungsi Global
parseFloat("10.5"); // 10.5
isNaN()

Fungsi Global
isNaN("Hello"); // true
isFinite()

Fungsi Global

eval()

Fungsi Global

decodeURI()

Fungsi Global

Fungsi Global
clearTimeout(timeoutId);
clearInterval()

Fungsi Global
clearInterval(intervalId);

üî¢ Math

Math.PI
Rasio keliling lingkaran terhadap diameternya, sekitar 3.14159
Properti
Math.PI; // 3.141592653589793
Math.E
Basis logaritma natural, sekitar 2.718
Properti
Math.E; // 2.718281828459045
Math.abs()
Mengembalikan nilai absolut dari angka
Metode
Math.abs(-5); // 5
Math.round()
Mengembalikan nilai pembulatan terdekat dari angka
Metode
Math.round(4.7); // 5
Math.ceil()
Mengembalikan bilangan bulat terkecil lebih besar atau sama dengan angka
Metode
Math.ceil(4.2); // 5
Math.floor()
Mengembalikan bilangan bulat terbesar kurang dari atau sama dengan angka
Metode
Math.floor(4.9); // 4
Math.random()
Mengembalikan angka pseudo-random antara 0 (inklusif) dan 1 (eksklusif)
Metode
Math.random(); // 0.123456789
Math.max()
Mengembalikan angka terbesar dari nol atau lebih angka
Metode
Math.max(1, 3, 2); // 3
Math.min()
Mengembalikan angka terkecil dari nol atau lebih angka
Metode
Math.min(1, 3, 2); // 1
Math.pow()
Mengembalikan basis dipangkatkan dengan eksponen
Metode
Math.pow(2, 3); // 8
Math.sqrt()
Mengembalikan akar kuadrat dari angka
Metode
Math.sqrt(16); // 4

üöÄ Fitur ES6+

let & const
Deklarasi variabel dengan scope blok (let bisa diubah, const tetap)
Fitur ES6
let x = 5; const y = 10;
Arrow Functions
Sintaks fungsi yang lebih ringkas dengan binding 'this' yang berbeda
Fitur ES6
const sum = (a, b) => a + b;
Template Literals
String literal yang memungkinkan ekspresi embedded
Fitur ES6
`Hello ${name}!`;
Destructuring
Ekstraksi data dari array atau objek ke variabel terpisah
Fitur ES6
const {name, age} = person;
Spread Operator
Memungkinkan ekspansi elemen iterable seperti array
Fitur ES6
[...arr1, ...arr2];
Rest Parameters
Merepresentasikan argumen tak terbatas sebagai array
Fitur ES6
function f(...args) { }
Default Parameters
Nilai default untuk parameter fungsi
Fitur ES6
function greet(name = "Guest") { }
Classes
Sintaks berbasis class untuk pembuatan objek dan pewarisan
Fitur ES6
class Person { constructor(name) { this.name = name; } }
Modules
Mengimpor dan mengekspor kode antara file JavaScript
Fitur ES6
import { func } from './module.js';
Promises
Objek yang merepresentasikan penyelesaian operasi asinkron
Fitur ES6
fetch(url).then(response => response.json());
async/await
Sintaks untuk bekerja dengan promises secara sinkron
Fitur ES7
async function getData() { const data = await fetch(url); }
